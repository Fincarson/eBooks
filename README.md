# 📚 eBooks

A curated collection of textbooks and study material, free to download.

Files are hosted as **GitHub Release assets**. Click any **⬇️ PDF** link and the download starts immediately. Because downloads come from Releases (not the file tree), links stay valid even if the catalog is reorganized.

- 🎓 [**Academic**](_docs/academic/README.md): 77 textbooks across 24 subjects
